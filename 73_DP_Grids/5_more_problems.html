Dynamic Programming Grid Problems

Easy
Foundation problems to build core concepts

1 Unique Paths

LeetCode

Find number of unique paths from top-left to bottom-right in a grid (only move right or down)

Why Easy: Basic 2D DP, simple recurrence relation: dp[i][j] = dp[i-1][j] + dp[i][j-1]
Solve Problem

2 Minimum Path Sum

LeetCode

Find a path from top left to bottom right, which minimizes the sum of all numbers along its path

Why Easy: Direct extension of unique paths with sum calculation
Solve Problem

3 Triangle

LeetCode

Given a triangle array, return the minimum path sum from top to bottom

Why Easy: 1D DP possible, clear adjacent movement rules
Solve Problem

5 Minimum Falling Path Sum

LeetCode

Return the minimum sum of any falling path through matrix. A falling path starts at any element in the first row and chooses the element in the next row that is either directly below or diagonally left/right

Why Moderate: 3 possible moves per step, requires handling edge cases for diagonal moves
Solve Problem

6 Maximum Path Sum in Grid

GeeksforGeeks

Find maximum sum path from first row to last row (can move diagonally)

Why Moderate: Multiple starting points, diagonal movements increase complexity
Solve Problem
